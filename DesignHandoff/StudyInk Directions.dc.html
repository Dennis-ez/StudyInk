Inkling Design Exploration

Three directions for a study tool you want to live in.

Each direction is a complete, opinionated system — not a recolor. They differ in typographic voice, density, shape language, how papery vs digital they feel, and how the AI tutor announces itself on the page. Below: hero mockups of the library and editor for each, the AI bubble (the heart), token swatches, and a one-line thesis. Pick a global winner — or mix surfaces across directions.

Selected direction

Foolscap — theme system

Six selectable themes. Each pairs your ink with your tutor's color, tints the chrome, and swaps the app icon — the paper never changes.

Every theme is the same token contract with different values, so it drops into one Swift enum. you.accent is your UI tint & default ink; ai.accent is the tutor's color. pagePaper and note thumbnails stay constant — only sidebar / chrome / editorDesk pick up the theme tint.

Mirroring rules
  • Sidebar → trailing edge; grid + content reflow to leading. Use leading/trailing, never left/right.
  • Selected-row accent bar, bubble tone strip, and speech tail all flip sides.
  • Directional glyphs mirror (undo/redo, send, chevrons); tool icons keep meaning.
  • Math & code render as LTR islands inside RTL prose; user ink direction follows the keyboard/pencil.
  • SwiftUI: .environment(\.layoutDirection, .rightToLeft) + SF Symbols auto-mirror.

System & engineering handoff

The cross-cutting system below is specified once and applies to every direction — only the token values differ per theme. Themes drop into a single Swift enum, each supplying a Palette; one reactive accessor resolves light/dark.

My recommendation

Ship Foolscap as the global base — it best matches “calm, warm, paper-like, focused” — and borrow Atelier’s Liquid-Glass material for the floating toolbar and AI bubble, where the iOS 26 frost genuinely earns its place.

Marginalia’s oversized serif library title is a great optional “focus” skin. The three share one token contract, so mixing is a values swap, not a rebuild. If you want maximum brand distinctiveness over calm, lead with Marginalia instead.
